# Generic Segment Tree

A generic segment tree implementation in C that supports adding, removing, updating, retrieving, and querying values using a custom comparator function.

## Features

- Generic storage using `void *`
- Dynamic resizing
- Range query support
- Custom comparator functions
- Error handling with `setjmp` / `longjmp`
- Interactive command-line driver
- Unit tests for segment tree operations
- Sorting utility using the segment tree
- Bungee Builder problem solver

## Technologies

- C
- Generic Data Structures
- Segment Trees
- Dynamic Memory Allocation
- Function Pointers
- setjmp / longjmp
- Command-Line Programs
- Unit Testing
