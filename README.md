# C Systems Programming Projects

A collection of systems programming and software tools projects implemented in C. These projects focus on parsing, dynamic memory allocation, data structures, cryptography, file processing, command-line utilities, and low-level systems concepts.

## Projects

---

## Project 1 — C Programming Fundamentals
Introductory C programs demonstrating formatted output, mathematical approximation, random text generation, and standard input/output processing.

### Concepts
- Standard I/O
- Loops and functions
- Taylor series approximation
- String formatting
- Input validation

---

## Project 2 — Tidy Data Cleaner
A command-line data cleaning and validation tool for parsing structured datasets and validating column formats.

### Features
- CSV-style dataset parsing
- Integer, floating point, and date validation
- Missing value handling
- Data normalization
- Error detection for malformed rows

### Concepts
- File I/O
- String parsing
- Data validation
- Modular programming

---

## Project 3 — Spreadsheet Formula Engine
A spreadsheet-style expression parser and evaluation engine capable of recursively evaluating formulas and resolving cell references.

### Features
- Recursive formula parsing
- Cell reference evaluation
- Arithmetic operations
- Overflow detection
- Divide-by-zero handling
- Table formatting

### Concepts
- Recursive descent parsing
- Expression evaluation
- Error handling
- File processing

---

## Project 4 — Vacation Rental Management System
A command-line reservation management system for vacation rental properties using dynamically allocated data structures.

### Features
- Reservation and cancellation management
- Searching and sorting listings
- Dynamic array resizing
- Interactive command handling
- File-based persistence

### Concepts
- Dynamic memory allocation
- Structs
- Function pointers
- Sorting with `qsort`
- Command parsing

---

## Project 5 — Triple DES File Encryption
A full implementation of Triple DES encryption and decryption for binary files.

### Features
- Triple DES encryption/decryption
- Bit-level permutation logic
- Key scheduling
- Padding support
- Binary file I/O
- Unit testing

### Concepts
- Cryptography
- Bitwise operations
- Block ciphers
- Memory management
- Systems programming

---

## Project 6 — Generic Segment Tree
A generic segment tree implementation supporting range queries, updates, dynamic resizing, and interactive command processing.

### Features
- Generic `void *` storage
- Range minimum/maximum queries
- Dynamic resizing
- Sorting utility
- Interactive command-line driver
- Unit testing
- Bungee Builder problem solver

### Concepts
- Generic data structures
- Function pointers
- Dynamic memory allocation
- `setjmp` / `longjmp`
- Range query algorithms
